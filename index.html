Welcome!

This website is the Lab Assignment 01 submission for the Human-Computer Interaction & Computer Graphics (HCI-CG) course. It demonstrates the practical application of semantic HTML, minimal CSS, accessibility features, and fundamental HCI models.

Navigation

  • Interaction Analysis: Detailed explanation of Norman, GOMS, and Suchman models and their application in website design.
  • HTML Errors: Common HTML mistakes with wrong and corrected code, demonstrating semantic HTML understanding.
  • HTML Quiz: MCQs, True/False, and Short Answer questions with answers for self-assessment.
  • References: Academic references used to build content, strictly following the HCI-CG textbook and original authors.

Purpose

The purpose of this assignment is to provide a semantic, accessible, and well-structured HTML website while learning:

  • Norman’s design principles (Visibility, Feedback, Mapping, Constraints, Affordances)
  • GOMS model (Goals, Operators, Methods, Selection rules)
  • Suchman’s situated action theory (adaptive interaction)
  • Minimal CSS styling for clarity and readability
  • Accessibility features like aria-current and alt text
