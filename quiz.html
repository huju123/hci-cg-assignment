Part A – Multiple Choice Questions

  1. Which element represents site navigation?
    a) div
    b) nav
    c) section
    d) span
    Answer: b
  2. Which attribute describes images for screen readers?
    a) src
    b) id
    c) alt
    d) class
    Answer: c
  3. Which model predicts execution time?
    a) Norman
    b) Suchman
    c) GOMS
    d) KLM
    Answer: d

Part B – True / False

  • Every page can have multiple h1 headings. – False
  • aria-current marks the active navigation link. – True
  • Tables should be used for layout. – False
  • GOMS analyzes goals and methods. – True
  • Suchman supports fixed interaction plans. – False

Part C – Short Answer

Q: What is semantic HTML?

A: Semantic HTML uses meaningful elements such as nav and header to describe content structure.

Q: Why does KLM ignore planning?

A: KLM models expert, error-free execution time only.

Q: What is Suchman’s main idea?

A: Human interaction is situated and adapts to context.
