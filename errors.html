1. Missing alt Attribute

Wrong:

<img src="photo.jpg">

Correct:

<img src="photo.jpg" alt="Student using laptop">

Images must include alt text for accessibility.

2. Multiple h1 Elements

Wrong:

<h1>Home</h1>
<h1>About</h1>

Correct:

<h1>Home</h1>
<h2>About</h2>

Only one h1 is allowed per page.

3. Link Outside li

Wrong:

<ul>
<a href="index.html">Home</a>
</ul>

Correct:

<ul>
<li><a href="index.html">Home</a></li>
</ul>

Links inside lists must be wrapped in li.

4. Label Not Connected

Wrong:

<label>Name</label>
<input id="name">

Correct:

<label for="name">Name</label>
<input id="name">

Labels must match input id.

5. Table Without Caption

Wrong:

<table>...</table>

Correct:

<table>
<caption>Student Marks</caption>
</table>

Tables need captions for screen readers.

6. Using div Instead of nav

Wrong:

<div>Menu</div>

Correct:

<nav>Menu</nav>

Semantic tags improve accessibility.

7. Skipping Headings

Wrong:

<h1>Title</h1>
<h3>Section</h3>

Correct:

<h1>Title</h1>
<h2>Section</h2>

Heading levels must not skip.

8. Missing aria-current

Wrong:

<a href="errors.html">Errors</a>

Correct:

<a href="errors.html" aria-current="page">Errors</a>

Active page must be indicated.

9. Button Without Type

Wrong:

<button>Submit</button>

Correct:

<button type="submit">Submit</button>

Button type avoids unexpected behavior.

10. Unclosed Tags

Wrong:

<p>Hello

Correct:

<p>Hello</p>

All tags must be properly closed.
